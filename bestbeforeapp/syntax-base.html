{ItemName}

Prints the name of the forwarding/sharing item.

Eg. Apple

{TimeLeft}

Prints the number of days before best before date.

Eg. 20 days

{BestBeforeDate}

Prints the best before date of the item in default format yyyy-MM-dd.

Eg. 2048-2-31

{BestBeforeDate:yyyy:MM:dd}

Prints the best before date of the item in format yyyy:MM:dd.

Eg. 2048:2:31

{Freshness}

Prints freshness of item.

Eg. Expired Recently

{Category}

Prints category of item.

Eg. Food

{Action}

Prints action to take for item.

Eg. Renew/Restock

{SomethingElse}

Prints the placeholder itself.

Eg. {SomethingElse}
