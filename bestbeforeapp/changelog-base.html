1.2.0

+ Sync

Finally we have sync. But sync is not fully tested, if you see any problem, you can turn it off in settings. And contact me if you don’t mind.

+ Forward Strategy

You can now set a forward strategy for a category. This part is rewritten, so you have to set the universal one again.

+ URL Scheme

Best Before now supports some URL Schemes

bestbefore:///item/:id/detail

bestbefore:///item/add bestbefore:///itemKind/:nameOrID/renew

There are something else, but I haven’t tested them out, so..maybe next version. Or you can change some words and see if that works.

Some URL schemes can be added to forward strategy, with {Action:Detail} and {Action:Renew}.

+ UI Update

Looks more modern. New icons. You can now see the category color in item lists. The pull-down-navigationbar thing is removed.

+ Some new bugs I have rewritten some logics again, if you see any of them, please let me know. Thanks for your support.
